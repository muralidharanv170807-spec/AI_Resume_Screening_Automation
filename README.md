AI Resume Screening Automation

An AI-powered recruitment automation system built using n8n, OpenAI, Gmail, Google Drive, and Google Sheets.

Features
- Automatically collects resumes
- Extracts resume text
- Uses AI to analyze candidate skills
- Scores candidates
- Stores results in Google Sheets
- Sends selection/rejection emails

Technologies
- n8n
- OpenAI
- Gmail API
- Google Drive
- Google Sheets
- JavaScript

How to Use
1. Import the workflow into n8n.
2. Connect your Gmail, Google Drive, and Google Sheets accounts.
3. Add your OpenAI API key.
4. Run the workflow.
